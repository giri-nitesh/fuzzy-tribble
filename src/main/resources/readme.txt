## Reading family tree 

Each line of input file would be in the form, except the first line which would
be the root of the family :: 

<Person2, Gender> <Person1> Relation of Person2 with Person1

We would assume that Person1 is already added to the family tree. Thereby maintaining
a chronological order of addition of family members

3rd Arguement is an integer:
1       HUSBAND,
2		WIFE,
3		SON,
4		DAUGHTER,
5		BROTHER,
6		SISTER,
7		PATERNAL_UNCLE,
8		MATERNAL_UNCLE,
9		PATERNAL_AUNT,
10		MATERNAL_AUNT,
11		SISTER_IN_LAW,
12		BROTER_IN_LAW,
